## General info
This Project is an API for company internal purchase order management system, which helps to manage the lifecycle of an order 

## Technologies
Project contains:
* Java version: 11
* Spring Boot version: 2.7.2
* Spring Data JPA
* Spring Web
* Spring Security
* MS SQL Server Driver
* Swagger2 version: 2.9.2
* Lombok Project
* Mockito
* JUnit5
* AssertJ
